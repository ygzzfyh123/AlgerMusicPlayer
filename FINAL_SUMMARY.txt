## 💡 技术亮点

### 1. 移动端独立方案架构

```
HTTP 请求
   ↓
[request.ts 拦截器]
   ↓
isMobileEnvironment() → true
   ↓
Standalone API 处理
   ├─ 检查缓存 → 命中则直接返回
   ├─ 生成 WEAPI 加密签名
   ├─ 直连网易云 API
   └─ 缓存结果
   ↓
返回数据
```

### 2. 内置加密算法完整实现

- ✅ AES-128-CBC 加密
- ✅ RSA 大数加密
- ✅ WEAPI 签名生成
- ✅ CryptoJS 库支持

### 3. 缓存系统

```typescript
// 自动为 GET 请求缓存 10 分钟
GET /search?keywords=xxx
  ↓
检查本地缓存 → 命中 → 直接返回（节省流量）
  ↓ 未命中
调用 API → 获取数据 → 本地缓存 → 返回

// 用户可配置缓存时间
mobileCache.setExpiry(5); // 改为 5 分钟
```
